Структура:
С начала объявление состояний
Потом команды

Идентификаторы:
    #
    Буква - тип задания(пока что он один)
    Буква - подтип логики задания:
        // Для типа "A":
          * A - задание
          * B - подсказка
    2 символа - номер задания (от 01 до 99)
    (
         //Для типа A:
           "
             Текст
           "
           ,
           "
             Код
           "
    )
    ;

Команды:
/
    BEGIN
    (
        [Идентификатор] - начальное состояние
    )
    ;

/
    END
    (
        [Идентификатор] - конечное состояние
    )
    ;

/
    JUMP
    (
        [Идентификатор] - от состояния
        ->
        [Идентификатор] - к состоянию
    )
    ;
